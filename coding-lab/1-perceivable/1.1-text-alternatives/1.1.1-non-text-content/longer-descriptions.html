A long description for this image. We dont hide this div with .sr-only since the more ways we present information to people the more inclusive the design is.

Notes on aria-describedby and aria-details

  • Any semantic in aria-describedby referenced element will be ignored by assistive technology
  • aria-details supports semantic in referenced element

longdesc attribute

  • Can point to seperate page or element on same page
  • Deprecated (search longdesc)

Best approach

aria-details or a normal link to a page with the long description seems like the best approach.
