My journey with a11y

Voice Over

Voice Over Key (VO) = CTRL + OPTION

Command Task
CMD + F5 Start / Turn Off
CTRL Stop Reading
VO + SHIFT + UP / DOWN Birds View Up / Drill Down
VO + LEFT / RIGHT Navigate Items on Current Level
VO + UP / DOWN Navigate Table Column
VO + Z Read Last Spoken Phrase
VO + A Read From Here
VO + P Read Paragraph
VO + S Read Sentence
VO + CMD + H (SHIFT reverse) Cycle Headers
VO + CMD + L (SHIFT reverse) Cycle Links
VO + CMD + T (SHIFT reverse) Cycle Tables
VO + CMD + J (SHIFT reverse) Cycle Form Controls
VO + CMD + X (SHIFT reverse) Cycle Lists
VO + SHIFT + I Read Info About Page
VO + U (Close with ESC) Roter

NVDA

NVDA Key = INSERT

Command Task
As a normal Windows application / NVDA + Q Start / Turn Off
CTRL Stop Reading
UP / DOWN Navigate Items
NVDA + UP Read Current Line
CTRL + UP / DOWN Read Prev / Next Paragraph
NVDA + DOWN Read From Here
NVDA + T Read Title
1-6 Header 1-6
H (SHIFT reverse) Cycle Headers
K (SHIFT reverse) Cycle Links
U (SHIFT reverse) Cycle Unvisited Links
V (SHIFT reverse) Cycle Visited Links
L (SHIFT reverse) Cycle Lists
NVDA + F7 (Close with ESC) Element List
